# YoutubeDownloader

O aplicativo baixa qualquer vídeo de qualquer tamanho no youtube, baixa no formato mp3 (áudio) ou no formato mp4 (vídeo)! A versão disponível hoje é a versão 1.0.0, o código é otimizado mas pode melhorar muito, e não temos ícones mas iremos providenciar em breve.

Para usar, basta ir em release, baixar o executável, instalar, colocar o link do vídeo que você deseja baixar, clicar no formato que deseja, e o download será iniciado para a pasta Downloads do seu computador
